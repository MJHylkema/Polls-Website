Overview

This app was made by myself, Mathew Hylkema for SENG365 assignment 2. It is a simple MVVM web application for viewing polls. It makes use of AngularJS to provide a interactive, responsive front end client and RESTful web services for all database related requests.

Features

  • View Polls
  • Vote on Polls and View Votes - Note: Link wont work if no polls in database.
  • Administer Polls - From here you can reset poll votes, and add and remove polls.
  • About Page

Styling

This website uses Bootstrap for almost all of it's css. I chose to use bootstrap as it provides a tidy design while being easy to implement. I implemented a background image to give the website a stylish look. For the navbar implementation, I based it off an example on the bootstrap website.

REST API

In all cases theprogram should respond with a 404 if something goes wrong or a 20X if everything is correct. I was able to implement poll creation and deletion as 'extra features' but did not get around to adding poll edit functionality.
The REST services provided by this program can be accessed by http://csse-studweb3.canterbury.ac.nz/~mjh300/365/polls/index.php/services/{query}
In the following section I will refer to this URI up until {query} as {BASEURI} as it is the base URI for all RESTful services.

Service URL Method Request_Data Response_Data Explanation
{BASEURI}/polls GET None 200 OK and a JSON list of polls. Each poll is a JSON object with an id, title, question and a list of answers Gets a list of all available polls
{BASEURI}/polls/{pollId} GET None 200 OK and a JSON object representing the poll with the specified id. JSON object consists of poll id, title, question and a list of answer objects - each consisting of an id, answer and associated poll id. If the id doesn't represent a valid poll a 404 will be returned. Gets a specific poll
{BASEURI}/polls POST A JSON object representing the poll to create 201 CREATED and a Location header with link to the new poll if the poll was successfully created or a 404 if something went wrong Creates a new poll
{BASEURI}/polls/{pollId} DELETE None 200 OK Deletes an existing poll.
{BASEURI}/votes/{pollId}/{answerNo} POST None 200 OK if the vote was successful or 404 if the poll or option does not exist Votes for an answer within a poll. {pollId} is the id of the poll to vote for and {answerNo} is the answer within the poll to vote for
{BASEURI}/votes/{pollId} GET None 200 OK and a JSON object consisting of a list of answers, each having an id, an answer, associated poll id and count - being the number of votes for this answer. Gets a list of all votes within a poll. {pollId} is the id of the poll to get votes for
{BASEURI}/votes/{pollId} DELETE None 200 OK Resets the votes on a poll. {pollId} is the id of the poll to reset
